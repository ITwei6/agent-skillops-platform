# Project Vision

## One-line Positioning

Agent SkillOps Platform is a team platform that converts individual agent work
experience into reviewed, versioned, searchable, and installable Codex skills.

## What This Is

This is a governance and asset platform around agents.

It manages the lifecycle of reusable team skills:

- Experience capture
- Skill proposal generation
- Automated safety and quality checks
- Duplicate detection
- Human technical review
- Versioned publication
- Team installation and usage feedback
- Outdated-skill detection

## What This Is Not

This is not a replacement for Codex, Cursor, Claude Code, LangGraph, or other
agent systems.

Those tools execute work. This platform captures, audits, and distributes the
valuable experience produced during that work.

## Main Users

- Developers who submit useful experience after finishing work with agents.
- Technical reviewers who approve or reject team skills.
- Team members who search and install approved skills.
- Team leads who want visibility into reusable engineering knowledge.

## Core Example

1. Developer A uses Codex to debug a C++ microservice deployment issue.
2. Developer A submits the summary, references, and a generated skill draft.
3. The platform checks whether the draft leaks secrets or duplicates an
   existing deployment skill.
4. A senior engineer reviews the proposal and requests a clearer trigger
   description.
5. After approval, the skill is published to the team registry.
6. Other developers install the skill when working on similar deployment tasks.
7. If the deployment docs change, the platform marks the skill as possibly
   outdated.

## Why It Matters

Agent answers are often temporary and scattered across personal chats. A team
needs durable engineering memory. This platform turns one-off agent outputs into
maintainable team assets.
